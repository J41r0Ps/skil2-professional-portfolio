## ✨ Key Features

* **Interactive 3D Experience:** Modern UI featuring 3D models integrated directly into the browser using React Three Fiber and Three.js.
* **Native Dark/Light Mode:** Seamlessly switch between themes with smooth transitions, built natively with Tailwind CSS.
* **Multilingual Support (i18n):** Fully accessible in **English, Dutch, Spanish, and French** to connect with a broader, international professional network.
* **Full-Stack Architecture (Phase 2):** A robust backend built with ASP.NET Core (.NET 8) serving dynamic data to a responsive React 19 frontend.
* **Automated DevOps:** Fully containerized with Docker and continuously deployed to Microsoft Azure via GitHub Actions CI/CD pipelines.

## 🛠️ Tech Stack Overview

* **Frontend (UI/UX & 3D):** React 19, Tailwind CSS, React Three Fiber (Three.js), Framer Motion, React-i18next.
* **Backend & API (Phase 2):** ASP.NET Core Web API (C#)
* **Database:** Containerized SQL Database
* **Infrastructure & DevOps:** Docker, Docker Compose, GitHub Actions, Microsoft Azure App Service

## 📂 Repository Structure

This repository follows a monorepo structure to keep the full-stack ecosystem unified and easily maintainable:

* `/.github/workflows` - CI/CD pipeline configurations for automated testing, building, and deployment to Azure.
* `/frontend` - React Single Page Application (SPA) containing all UI components, 3D Canvas logic, and client-side routing.
* `/backend` - ASP.NET Core Web API handling data persistence and business logic (Upcoming Phase).
